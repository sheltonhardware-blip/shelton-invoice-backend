### **To Update Contact Information:**

Edit `business-card/index.html`:

```html
<!-- Line 208: Name -->
<div class="person-name">Your Name</div>

<!-- Line 209: Title -->
<div class="person-nick">Your Title</div>

<!-- Update phone, email, address in contact section -->
```

### **To Change Colors:**

Edit CSS variables (around line 50):

```css
:root {
    --gold: #YOUR_COLOR;      /* Primary color */
    --ink: #0D0D0D;            /* Background */
    --cream: #F8F3EC;          /* Card background */
}
```
